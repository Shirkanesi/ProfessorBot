# ProfessorBot
Der Professor-Bot ist ein Open-Source-Projekt, das bedeutet, dass jeder (der Lust und Zeit hat und obendrein das Chaos auch noch versteht) helfen kann, diesen Bot zu verbessern, wobei das ja kaum möglich ist. :D

_Butterbrot als Basis_

Dieser Bot basiert ursprünglich auf Butterbrotbot von Ypsilondev, wurde aber für unsere Zwecke komplett angepasst
